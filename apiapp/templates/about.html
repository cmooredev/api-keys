How it works

The bot uses multiple translation services to get the most accurate result. First we detect the language and then route it through either deepL or Google translate, two of the most accurate translation services.

Learn more

Adding the bot to your server.

After purchase, you will be provided a link to add the bot to your server. Make sure the server id matches the one you provided during purchase. Once the bot is added to your server, you can use the command .config to set your target language. After it is set, the bot will translate all users with the role 'Translate' into the set target language.

Commands

.trconfig : Bring up menu to select language.

.trstats : Check current language and available credits.

Commands in development

.trconfig @user: Select specific language to translate user into.

New bots are being added regularly. Contact us for custom requests.
