How it works

The bot uses multiple translation services to get the most accurate result. First we detect the language and then route it through either deepL or Google translate, two of the most accurate translation services. If you are on the free plan, we run the message through a free machine translation service. This has feature unlimited, free use!

Adding the bot to your server.

After purchase, you will be provided a link to add the bot to your server. Make sure the server id matches the one you provided during purchase.

Once the bot is added to your server, users can use the command .setlang to set their target language. This will also create a 'Translate' role and give it to person who used the command.

Commands

.setlang : sets language for using interacting with select menu. This will also give you the Translate role. To stop translations, remove this role.

.stats : Check current credits for paid services.

.untranslate : Remove the translate role.

Users must have the translate role in order for the bot to recognize them.

New bots are being added regularly. Contact us for custom requests.
